# Référence visuelle — Agence L&L

`accueil-prototype.html` est la **maquette d'accueil validée** avec Julien & Zoé
(28/06/2026). C'est la **direction esthétique de référence** : reproduis-la fidèlement
en Astro (composants + pages), elle fait foi sur la palette, la typo (Fraunces/Hanken/
Mono/Caveat), le sceau-pieuvre, les numéros de projet manuscrits rouges, le rythme
éditorial asymétrique et les repères d'impression.

Ouvre-la dans un navigateur pour voir le rendu cible. Le sceau y est déjà le vrai SVG
(repris dans `src/components/Monogram.astro`). Les **textures de matière** (pisé,
chaux-chanvre, bois) y sont **temporaires** : elles seront remplacées par les vraies
photos de chantier.
